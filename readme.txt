== Description ==

This is a highly customizable jQuery Slider plugin. You can set its width, height, pagination and other
parameters. You can use it on your post or page via a schortcode `[jQuery Slider]` or in your template 
file  using php shortcode`<?php echo do_shortcode('[Gallery]') ?>`. or in editor [Gallery]

Slides can be added as post type slide from slides menu.

Tags: jQuery, Slider, Gallery, Plugin, Widget

Tested up to: 3.3.2

You can see its working demo here: http://www.wp-themeforest.com/rotating-lightbox-gallery/

== Installation ==

1. Download the plugin
2. Upload the `rotation-Lightbox-gallery` folder to the `/wp-content/plugins/` folder
3. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. This is how the slider will look on you post or page.
2. Go to Gallery menu /wp-admin/edit.php?post_type=gallery
3. Click on Add New button wp-admin/post-new.php?post_type=gallery. You can set its Title and Image
image via posts content area and featured image section.
